Designer principal + revisão por lead

🟢 Organisms: ProductCard, Header, FilterSidebar

🔘 Todos os botões (primary, secondary, danger)

3
Auto Layout Obrigatório
Responsável:
Todos os designers | Prazo: 100% dos containers
Aplicar Auto Layout em:
📦 Todos os containers de produtos
📋 Listas e grids
📝 Formulários completos
🧭 Barras de navegação
🎴 Layouts de cards
📁 Estruturas de menu
⚠️ TESTE OBRIGATÓRIO:
Redimensione o frame no Figma - deve comportar como esperado no código final
4
Design Tokens/Variables
Responsável:
Design lead + aprovação do time
Prazo: Configurar semana 1, aplicar em todos designs
🎨 Cores Obrigatórias:
Colors/Primary/500 → --color-primary
Colors/Secondary/500 → --color-secondary
Colors/Success/500 → --color-success
Colors/Error/500 → --color-error
Colors/Price/500 → --color-price
📏 Espaçamentos (grid 8px):
Spacing/XS → 4px → --space-1
Spacing/SM → 8px → --space-2
Spacing/MD → 16px → --space-4
Spacing/LG → 24px → --space-6
Spacing/XL → 32px → --space-8
📝 Typography:
Text/Heading/Large → --font-size-xl
Text/Body/Regular → --font-size-base
Text/Caption/Small → --font-size-sm
5
Annotations Comportamentais
Responsável:
Designer do componente | Prazo: Durante criação do design
🔍 Documentar obrigatoriamente:
🎯 Estados hover/focus/active
📱 Comportamento responsivo específico
🔄 Interações complexas
⏳ Estados loading/error/empty
✨ Transições e animações
📝 Exemplos de Annotations:
"Botão fica desabilitado quando produto out-of-stock"
"Card tem hover effect com shadow elevation"
"Em mobile, sidebar vira drawer bottom-sheet"
"Loading state mostra skeleton por 2s"
